## Purpose

Thư viện hành vi evidence-based với:

- Tìm kiếm theo từ khóa (10-15 keywords/hành vi)
- Phân nhóm theo lý thuyết hành vi
- Yêu thích và xu hướng
- Chỉ số nghiên cứu khoa học (🎓)

## Components

### Header

- Evidence-based badge (🎓)
- Total behavior count
- Subtitle: "Evidence-based • 127 hành vi"

### Search & Filters

- Search bar với gợi ý keywords
- Filter by group (Nhóm)
- Sort options
- Favorites filter

### Favorites Section

- Hierarchical IDs (1.1, 1.2, 2.1)
- Vietnamese & English names
- Group classification
- System usage count
- Student-specific count
- Evidence-based badge (🎓)

### Trending Section

- Weekly trends
- Change percentage
- Trend indicators
- Behavior IDs

### Behavior Groups Section

**New Structure:**

- Replaces old 4-category grid
- Groups based on theoretical frameworks:
  1. **CHỐNG ĐỐI & BƯỚNG BỈNH** (Opposition & Defiance)
     - Icon: 😤
     - Count: 2 behaviors (1.1, 1.2)
  2. **HÀNH VI GÂY HẤN** (Aggression)
     - Icon: 👊
     - Count: 1 behavior (2.1)
  3. **VẤN ĐỀ VỀ GIÁC QUAN** (Sensory Issues)
     - Icon: 👂
     - Count: 1 behavior (3.1)
- Each group shows evidence-based badge
- Behavior count per group

### Data Structure Enhancements

**Behavior Card Now Includes:**

- **ID**: Hierarchical numbering (e.g., 1.1, 1.2)
- **Name**: Bilingual (Vietnamese + English)
- **Group**: Theoretical classification
- **Evidence Badge**: 🎓 indicates research-backed
- **Stats**: System-wide + personal usage

**Keywords for Backend Search (not displayed to users):**

Each behavior has 10-15 Vietnamese keywords used for filtering and search functionality:

- Example (Ăn vạ): "ăn vạ", "la hét", "nằm lăn ra đất", "gào khóc", "tức giận dữ dội"
- Example (Từ chối): "không nghe lời", "không làm theo", "phớt lờ", "kháng cự"
- Example (Đánh bạn): "đánh bạn", "đẩy bạn", "xô ngã", "cấu", "véo"

These keywords enable robust full-text search but are NOT shown in the UI.

## Interactions

- **Search by keywords** = Tìm theo 10-15 từ khóa/hành vi
- **Filter by group** = Lọc theo nhóm lý thuyết
- **Tap behavior** = View detail with full data
- **Tap star** = Toggle favorite
- **Tap group** = View all behaviors in group
- **Tap "Xem tất cả"** = Full list with filtering
