## Purpose

Quản lý thông tin cá nhân và cài đặt tài khoản.

## Components

### Avatar Section

- Profile photo (100x100)
- Name
- Edit photo button

### Personal Info

- Email (editable)
- Phone (editable)
- School/Organization (editable)

### Security

- Change password
- Two-factor authentication

### Data Management

- Backup status & action
- Export data (CSV/PDF)

### Support

- User guide
- Contact support
- Rate app

### Logout

- Sign out button

### Footer

- Version number
- Copyright

## Interactions

- **Tap avatar** = Change photo
- **Tap Sửa** = Edit field
- **Tap Đổi mật khẩu** = Change password flow
- **Tap Bật 2FA** = Enable 2FA
- **Tap Sao lưu** = Backup data
- **Tap Xuất** = Export options
- **Tap Đăng xuất** = Confirm & logout
